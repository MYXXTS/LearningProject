# 05_bike_lock_controller

当前整机固件工程，适用于 `STC89C52RC`。

## 引脚分配

- `P0.0-P0.7`：LCD1602A 数据总线
- `P2.5`：LCD `RW`
- `P2.6`：LCD `RS`
- `P2.7`：LCD `EN`
- `P2.0`：SG90 舵机 PWM
- `P2.1/P2.2`：GPS 保留接线，不参与整机流程
- `P2.3`：MP3 软件 UART `TXD`
- `P3.0`：HC-06 `RXD`
- `P3.1`：HC-06 `TXD`

## 当前行为

- HC-06 使用硬件 UART，`9600 8N1`
- 舵机使用 `Timer0`
- MP3 使用 `Timer2` 自动重装的软件 UART 发送
- 发送 MP3 指令时，会短暂停掉蓝牙接收与舵机中断，避免位时序被打断
- GPS 保持接线，但整机固件不读取 GPS 数据

## 支持协议

- `U`：开锁，返回 `U,OK` / `U,ALREADY` / `U,BUSY`
- `L`：关锁，返回 `L,OK` / `L,ALREADY` / `L,BUSY`
- `S`：查询状态，返回 `S,0` / `S,1`
- `P,1..5`：播放提示音，返回 `P,OK`
- 非法曲目：返回 `P,BAD`
- 非法文本：返回 `E,BAD`

支持的结束符：

- `\n`
- `\r`
- `\r\n`

另外，`U` / `L` / `S` / `P,1` 这类固定长度命令也支持无换行直接识别。

## 曲目映射

- `001`：开锁成功提示音
- `002`：开锁重复/失败提示音
- `003`：关锁成功提示音
- `004`：关锁重复/失败提示音
- `005`：围栏拒绝提示音
